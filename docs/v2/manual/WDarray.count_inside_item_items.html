WDarray: size and location

WDarray.count

It is a method that returns the amount of times the information passed to the argument appears in the array.

wd(array).count(arg)

Example

var y = wd();
y.count(wd);

y.count(false);

WDarray.inside

It is a method that returns true if the argument entered is contained in the array, otherwise returns false. There is a second optional argument which, if true, will return an array indicating the indexes in which the information appears within the array.

wd(array).inside(arg [,show = false])

Example

//Example continuation
y.toString();

y.inside(wd);

y.inside(DW);

y.inside(wd, true);

WDarray.item

It is a method that returns the value of the index entered in the argument. the value of the argument corresponds to the index of the array. Negative numbers return values ​​from the end of the array. For example, -1 returns the last item in the array and -2 the penultimate.

wd(array).item(index)

Example

//Example continuation
y.toString();

y.item(2);

y.item(-1);

y.item(-3);

WDarray.items

Is an attribute that returns the length of the array.

wd(array).items

Example

//Example continuation
y.toString();

y.items;
