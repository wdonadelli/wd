WDarray: organizing items

WDarray.sort

It is a method that returns the ordered array in an increasing manner obeying the following sequence of data types: null values, numbers, time, date, text and other information. There is an argument that, if true, excludes repeated items in the same way as the unique method.

wd(array).sort(unique)

Example

var x = ;
var y = wd(x);
y.sort();

y.sort(true);

WDarray.unique

It is a method that returns the array without repeating items. There is an argument that, if true, sorts the array items in the same way as the sort method.

wd(array).unique(sort)

To be disregarded, repeated items must be in the same format, ie 1 (String) will be different from 1 (Number), so they will be considered different and will be kept in the array.

Example

var x = ;
var y = wd(x);
y.unique();

y.unique(true);
