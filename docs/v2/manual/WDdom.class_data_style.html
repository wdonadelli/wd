WDdom.class

It is a method that manipulates the class attribute.

wd(dom).class(list)

The list argument is an object that accepts three keys: add, del and toggle, executed in this order. The key values ​​are strings that represent the value to be added, removed, or alternate, respectively. If there are more than one attribute to set, the values ​​must be separated by white space. If the value is null the value of the class attribute will be empty.

WDdom.data


		
It is a method that that manipulates the data attribute.


		
wd(dom).data(list)


		
The list argument is an object whose keys and values ​​respectively represent the names of attributes and values. Do not use the data- prefix in the attribute name. Compound attributes can be represented by dashes or camelCase. If the key value is null, the value of the attribute is removed from the element. If the list argument is null, all keys in the dataset attribute are cleared.


		
This method is an important tool for defining the attributes present in WD Attributes Tools.

WDdom.style


		
It is a method that manipulates the style attribute.


		
wd(dom).style(list)


		
The list argument is an object whose keys and values ​​represent, respectively, CSS attributes and values. Keys can be in CSS or camelCase style. If the value is null the value of the style attribute will be empty. If the attribute does not exist, it can be ignored..


		
Example


		
HTML code


		


		
JavaScript
