WD Attributes Tools consists of a JavaScript library that automates some methods of the WD JavaScript Tools HTML object as attributes to be inserted into HTML elements.

Its syntax is as follows:

data-wd-attributeName=value

The attributeName defines the name of the attribute that has the following possibilities:

  • load
  • repeat
  • sort
  • filter
  • mask
  • page
  • action
  • sort-col
  • desktop
  • tablet
  • phone

Each attribute is sensitive to a certain event, whose possibilities are listed below:

  • onLoad
  • onResize
  • onClick
  • onKeyUp
  • onHashChange
  • onChange (see HTML.data object)
  • onDefinition (see HTML.data object)
//Example
<a href=# data-wd-action=hide>Hide</a>
