This is an object for manipulating strings. The supplied argument (primitive value) is not changed by methods or attributes. The following primitive values ​​define the object:

  • Strings not specified.

Strings can specify dates, time, numbers, and accessible paths. The object that this section deals with will be displayed when the other quoted string forms are not specified. As for accessible paths, they inherit the same methods / attributes of this object.

When identifying the type of the primitive value, the object returns the string text.

//Example
var input = Deus lux mundi est.;
var object = wd(input);
object.type;
text

The object has the following attributes:

  • title
  • trim
  • tt

The object has the following methods:

  • mask (Deprecated - js.v.1.3.0 - 2019-03-24)
