WD JavaScript Tools

The WD JavaScript Tools consists of a JavaScript function that, depending on the type and content of the passed argument, returns an object with specific attributes and methods.

Its syntax is as follows:

wd(input)

The input value defines the object to return and its attributes/methods. The object returned is organized into seven categories:

  • Array
  • Date
  • Time
  • HTML
  • Number
  • RegExp
  • Text

See the menu for more details.

These objects inherit their basic characteristics from a common object that is invoked when the information passed in the argument does not fall into any of the categories mentioned. The following characteristics are shared:

type

It is an attribute that identifies the type of information passed on. The following values strings ​​can be returned:

  • undefined
  • null
  • boolean
  • number
  • date
  • time
  • array
  • regexp
  • function
  • dom
  • text

If the argument does not fit into any of the above categories, it returns the constructor name. And if there is no constructor, it will return unknown.

Example

wd().type;

Tools

Is an attribute that returns a list of enumerable methods and attributes of the invoked object.

Example

wd().tools;

valueOf and toString

These are methods that return object comparison (valueOf) and text (toString) values. They may or may not accept arguments and return different types of values ​​depending on the type of object invoked. Inexisting specific value, its default value is the one defined by the browser.

Example

wd().toString();

wd().valueOf();

wd(null).toString();

wd(null).valueOf();

wd(true).toString();

wd(true).valueOf();

wd(false).toString();

wd(false).valueOf();
