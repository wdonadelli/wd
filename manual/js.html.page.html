This is a method that divides the child elements into defined quantity groups and display a given group, leaving the others inhibited. It is called as follows:

wd(input).page(page, size);

The page argument is a Integer that indicates the group to be displayed. Zero indicates the first group, 1 the second group, -1 the last group, -2 the penultimate group, and so on. If the reported value exceeds the number of groups, the threshold group is displayed, that is, the first or last group, as the case may be..

The size argument is Integer that indicates the size of each group. If the number is greater than zero and less than or equal to one, the size of the group will be defined by percentage.

//Example
function f() {
	var page = document.getElementById(js_html_page_example1).value;
	var size = document.getElementById(js_html_page_example2).value;
	wd$(#js_html_page_example2).page(page, size);
};
