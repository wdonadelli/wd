WDdate: setting date

WDdate.day

It is a attribute that that gets/sets the value of the day (1-31). If you set a value beyond the limits, the date will be adapted.

wd(date).day

Example

var x = ;
var y = wd(x);
y.day;

y.day = 21;
y.day;

y.day = 0;
y.day;
