CSS: Form

These are classes for styling form elements. It does not have the suffix -child.

There are three types of styling for form elements:

  • wd-label
  • wd-input
  • wd-button

There are two possibilities for styling for forms:

  • wd-vform (vertical)
  • wd-hform (horizontal)

The types radio and checkbox must be within a label element. In the color input type, the wd-button class should be used instead of wd-input.
