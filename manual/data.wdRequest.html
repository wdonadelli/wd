dataWdRepeat

The purpose of the attribute is to clone the child elements from a JSON format file to the parent element that contains the attribute.

See the following chapters:

  • WDdom: content (WDdom.repeat)
  • WDtext: request

Attribute

For the POST method (default):

data-wd-repeat=post{JSON file path}

For the GET method:

data-wd-repeat=get{JSON file path}

Target

  • Child elements.

Sensitivity

  • After loading the document;
  • After defining the attribute on the element with the method WDdom.data;
  • After external data loading procedures using the WD Attributes Tools library.
