dataWdRequest

The purpose of the attribute is to make requests and make it possible to work with the response.

See the following chapters:

  • WDtext: request

Attribute

For the POST method (default):

data-wd-request=post{file path}method{function}

For the GET method:

data-wd-request=get{file path}method{function}

The function name defined in the method subaction must be within the main scope of the document (window).

Sensitivity

  • When the element receives a click.
