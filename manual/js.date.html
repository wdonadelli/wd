This is an object for date manipulation, disregarding the effects inherent to the time zone. The time record can be changed by methods and attributes, the original value is not changed. The following primitive values ​​define the object:

  • YYYY-MM-DD
  • MM/DD/YYYY
  • DD.MM.YYYY
  • Date Object
  • %today

Where:

YYYY:
Four-digit year (integer from 0001 to 9999).
MM:
Month with two digits (integer from 01 to 12).
DD:
Day with two digits (integer from 01 to 31).
%today:
Current date.

When identifying the type of the primitive value, the object returns the string date.

//Example
var x = wd(01/01/2010);
x.year;
2010
x.year = 2011;
x.year;
2011
x.toString();
2011-01-01
x.toSource();
01/01/2010
x.type;
date

The object has the following attributes:

  • week
  • day
  • month
  • year
  • leap
  • days
  • width
  • weeks
  • countdown

The object has the following methods:

  • toString
  • valueOf
  • format
