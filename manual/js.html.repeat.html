This is a method that clones child elements from an array of objects using the object's keys as a parameter to define the characteristics of each cloned element. To initiate cloning, a child model element must be defined before executing the method. The number of clones will equal the number of items in the array. The characteristics of the clones must be within {{key}}, whose value will be captured from the reported object. It is called as follows:

wd(input).repeat(obj);

The obj argument is the an array of objects. Each item in the array will be an object whose keys and values ​​will define the characteristics of the element.

//Example
var x = [
	{nominativo: nox, genitivo: noctis},
	{nominativo: rex, genitivo: regis},
	{nominativo: lex, genitivo: legis},
	{nominativo: pax, genitivo: pacis}
];

var y = [
	{nominativo: noctes, genitivo: noctium},
	{nominativo: reges,  genitivo: regum},
	{nominativo: leges,  genitivo: legum}
];

function f() {
	wd$(#js_html_repeat_example1).repeat(x);
};

function g() {
	wd$(#js_html_repeat_example1).repeat(y);
};
