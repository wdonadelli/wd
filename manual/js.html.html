This is an object for manipulation of DOM/BOM. The supplied argument (primitive value) is not changed by methods or attributes. The following primitive values ​​define the object:

  • HTMLElement
  • NodeList
  • HTMLCollection
  • HTMLAllCollection
  • HTMLFormControlsCollection
  • document
  • window

When identifying the type of the primitive value, the object returns the string html, if a single element, [html], if a list of elements, doc, if a document object, or win, if a window object.

The methods of the object return to itself, so it is possible to apply them in sequence. The attributes return a list and each item in the list contains the requested product for each element indicated in the input argument.

//Example
var x = document.querySelectorAll(div);
var y = document.getElementById(ID1);
wd(x).type;
[html]
wd(y).type;
html
//methods in sequence
wd(x).action(open).filter(target);
//first item of x
wd(x).get[0];
<div#div1>

If you use the querySelectorAll method, you can use the object wd$ instead of wd. It is called as follows:

wd$(css, root);

The css argument must be a CSS selector and the root argument (optional) must be a parent element, its default value is document.

The function argument .

//The same result is obtained:
wd(document.querySelectorAll(div));
wd$(div);
//or...
document.getElementById(data1).querySelectorAll(div);
wd$(div, document.getElementById(data1));

  • get
  • styles

The object has the following methods:

  • handler

  • filter
