This is a attribute that returns an array whose items are objects that bring information about each element.

Objects have two attributes: the default, which gives information about the element's default styles, and the computed, which gives information about the computed styles of the element. The result will depend on the implementation of methods getComputedStyle and getDefaultComputedStyle by the browser. It is called as follows:

wd(input).styles;
