This is a method that returns a string with local numeric formatting, if the browser contemplates the corresponding method. Otherwise, it returns the number entered in string format. It is called as follows:

wd(input).locale(cod);

The cod argument (optional) consists of a string representing the location (e.g. pt-BR). If not set, the location will be the one entered in the lang attribute of the html tag, the value provided by the browser or en-US.

//Example
wd(1000.23).locale(pt-BR);
1.000,23
wd(1000.23).locale(en-US);
1,000.23
