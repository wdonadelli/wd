This is an object for manipulating regular expressions. The supplied argument (primitive value) is not changed by methods or attributes. The following primitive values ​​define the object:

  • Regular Expression

When identifying the type of the primitive value, the object returns the string regex.

//Example
var input = [/[^0-9]/];
var object = wd(input);
object.type;
regex
