WDarray: changing items

WDarray.add

It is a method that adds arguments to the end of the array and returns its new value.

wd(array).add(arg1, arg2, arg3...)

Example

var x = ;
var y = wd(x);
y.add(1, 2, 3, 3);

wd(x).toString(); //does not change the original object

y.toString();

WDarray.del

It is a method that removes all elements from the array that are identical to the arguments given and returns their new value.

wd(array).del(arg1, arg2, arg3...)

Example

//Example continuation
y.toString();

y.del(wd, 1);

WDarray.toggle

It is a method that removes or adds elements to the array, depending on its existence in the list, and returns its new value. If the passed argument is in the array, it will be deleted, otherwise it will be added to the end.

wd(array).toggle(arg1, arg2, arg3...)

Example

//Example continuation
y.toString();

y.toggle(wd, 100);

y.toggle(wd, 100);

WDarray.replace

It is a method that replaces the information contained in the first argument (old) with the information contained in the second argument (new) and returns its new value. All identical elements located in the array will be replaced.

wd(array).replace(old, new)

Example

//Example continuation
y.toString();

y.replace(3, 30);
