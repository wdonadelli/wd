This is a attribute that gets the time value (12h clock). It is called as follows:

wd(input).ampm;
//Example
var x = wd(22:24);
x.ampm;

This is a method that sets time format return. It is called as follows:

wd(input).format(string);

The string argument (optional), if not defined, will return the same object result with the toString method. The argument consists of a string where you can enter attributes of the object as format below:

%h:
Numerical value of the hour.
%H:
Numerical value of the hour (24h clock).
#h:
Time am/pm format.
#H:
Numerical value of the hour (24h clock, two digits).
%m:
Numerical value of the minutes.
%M:
Numerical value of the minutes (two digits).
%s:
Numerical value of the seconds.
%S:
Numerical value of the seconds (two digits).
//Example
var x = wd(25:05); //25 = 24h (00h) + 1h
var f = %H hour(s) and %m minute(s);
x.format(f);
