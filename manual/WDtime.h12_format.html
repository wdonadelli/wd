WDtime: formats

WDtime.format

It is a method that returns the tempo of the preformatted object.

wd(time).format(string, locale)

The string argument (optional), if not defined, will return the same object result with the toString method. The argument consists of a string where you can enter attributes of the object as format below:

%h
Numerical value of the hour.
%H
Numerical value of the hour (two digits).
#h
Time in 12-hour format.
%m
Numerical value of the minutes.
%M
Numerical value of the minutes (two digits).
%s
Numerical value of the seconds.
%S
Numerical value of the seconds (two digits).

The locale argument (optional) consists of a string representing the location (e.g. pt-BR). If not set, the location will be the one entered in the lang attribute of the html tag, the provided browser value or en-US

Example

wd(09:06:31).format();

WDdate.h12

This is a attribute that gets the time in 12-hour format.

wd(time).h12

Example

wd(22:24).h12;

wd(10h24).h12;
