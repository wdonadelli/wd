WDdom.repeat


		
It is a method that clones child elements from an array of objects using the object's keys as a parameter to define the characteristics of each cloned element. To initiate cloning, a child model element must be defined before executing the method. The number of clones will equal the number of items in the array. The characteristics of the clones must be within {{key}}, whose value will be captured from the reported object.


		
wd(dom).repeat(array)

		
		
The array argument is an array of objects. Each item in the array will be an object whose keys and values ​​will define the characteristics of the element.
