This is a attribute that gets/sets the value of the month from 1 (January) to 12 (December). If you set a value beyond the limits, the time will be adapted. It is called as follows:

wd(input).month;
//Example
var x = wd(2010-01-01);
x.month;
1
x.month = 6;
x.month;
6
x.month += 12;
x.month;
6
x.toString();
2011-06-01
