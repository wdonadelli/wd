WDdom.action

It is a method that defines actions for the element.

wd(dom).action(act)

The act argument is a string that represents the behavior to be applied to the element, as shown below:

open
If the element contains the open attribute, it will be set to true. Otherwise, the wd-open class will be added to the element becoming dependent on the style sheet.
close
If the element contains the open attribute, it will be set to false. Otherwise, the wd-open class, if any, will be removed from the element.
toggle-open
Toggles between open and close.
tab
Displays only the element informed, your brothers will be hidden.
del
Removes the element from DOM.
show
Shows the element.
hide
Hides the element.
check
Check input elements of type radio or checkbox.
uncheck
Uncheck input elements of type radio or checkbox.
toggle-check
Toggles the check of the input elements of the type radio or checkbox.
clean
Clears the content of the element.

WDdom.filter


		
It is a method that that displays only child elements that display the textual content entered in the argument.


		
wd(dom).filter(text, min, show)


		
The text argument is the text to be located in the element.


		
The min argument (optional) is the minimum number of characters for the method to be executed. The default value is zero.


		
The show argument (optional) is a Boolean value and defines whether the elements to be found will be displayed as long as the minimum number of characters is not reached. The default value is true.

WDdom.page


		
It is a method that divides the child elements into defined quantity groups and display a given group, leaving the others inhibited.


		
wd(dom).page(page, size)


		
The page argument is a integer that indicates the group to be displayed. Zero indicates the first group, 1 the second group, -1 the last group, -2 the penultimate group, and so on. If the reported value exceeds the number of groups, the threshold group is displayed, that is, the first or last group, as the case may be.


		
The size argument is Integer that indicates the size of each group. If the number is greater than zero and less than or equal to one, the size of the group will be defined by percentage.

WDdom.sort


		
It is a method that sorts the child elements in ascending or descending order.


		
wd(dom).sort(order, col)

		
		
The order argument (optional) is the number that indicates the order. If less than zero, decreasing, otherwise increasing. The default value is 1.


		
The col argument (optional) is the number that indicates the column to use as a reference to define the order. If the child elements, which are to be sorted, also have children (columns), it is possible to define these elements as a parameter for sorting (eg td elements of a table). The default value is null.
