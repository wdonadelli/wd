This is a method that manipulates the data attribute. It is called as follows:

wd(input).data(list);

The list argument is an object whose keys and values ​​respectively represent the names of attributes and values. Do not use the data- prefix in the attribute name. Compound attributes can be represented by dashes or camelCase. If the key value is null, the value of the attribute is removed from the element.

This method is an important tool for defining the attributes present in WD Attributes Tools.

Example

var x = document.querySelectorAll(a);
wd(x).style({
	wdSort: 1,              //camelCase: data-wd-sort=1
	wd-load: data.json, //Dash: data-wd-load=data.json
	example: null           //removing attribute data-example
});
