This is a attribute that gets/sets the value of the year (1-9999). If you set a value beyond the limits, the time will be adapted. It is called as follows:

wd(input).year;
//Example
var x = wd(2010-01-01);
x.year;
2010
x.year = 1724;
x.year;
1724
x.year = 99999;
x.year;
9999
x.toString();
9999-12-31
