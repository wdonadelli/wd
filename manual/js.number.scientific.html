This is a method that returns a string, in html format, in scientific notation (e.g. 1.00 x 102). It is called as follows:

wd(input).scientific(n);

The n argument indicates the number of decimal places to display in the notation.

//Example
wd(9876).scientific();
9.876 x 10<sup>3</sup>
wd(9876).scientific(1);
9.9 x 10<sup>3</sup>
wd(-1/9876).scientific(2);
-1.01 x 10<sup>-4</sup>
