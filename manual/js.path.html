This is an object that executes and works with asynchronous requests. The supplied argument (primitive value) is not changed by methods or attributes. The following primitive values ​​define the object:

  • A file or path accessible on the server.

When identifying the type of the primitive value, the object returns the string path.

//Example
//"file.json" is an accessible file on the server:
var object = wd(file.json);
object.type;
path

The object has the following methods:

  • get
  • post
