WDdate: formats

WDdate.format

It is a method that returns the data of the preformatted object.

wd(date).format(string, locale)

The string argument (optional), if not defined, will return the same object result with the toString method. The argument consists of a string where you can enter attributes of the object as format below:

%d
numerical value of the day.
%D
numerical value of the day (two digits).
@d
day of year.
%m
numerical value of the month.
%M
numerical value of the month (two digits).
@m
number of days in month.
#m
short name of the month.
#M
long name of the month.
%y
numerical value of year.
%Y
numerical value of year (four-digit).
%w
numerical value of day of week.
@w
numerical value of week of year.
#w
short name of the day of week.
#W
long name of the day of week.
%l
days in the year.
%c
days remaining for the end of the year.

The locale argument (optional) consists of a string representing the location (e.g. pt-BR). If not set, the location will be the one entered in the lang attribute of the html tag, the provided browser value or en-US

Example

wd(%today).toString();

wd(%today).format();

WDdate.longMonth

It is a method that returns the long name of the month.

wd(date).longMonth(locale)

The definition of locale is the same as the format method.

Example

wd(%today).longMonth();

WDdate.longWeek

It is a method that returns the long name of the day of week.

wd(date).longWeek(locale)

The definition of locale is the same as the format method.

Example

wd(%today).longWeek();

WDdate.shortMonth

It is a method that gets the short name of the month.

wd(date).shortMonth(locale)

The definition of locale is the same as the format method.

Example

wd(%today).shortMonth();

WDdate.shortWeek

It is a method that returns the short name of the day of week.

wd(date).shortWeek(locale)

The definition of locale is the same as the format method.

Example

wd(%today).shortWeek();
