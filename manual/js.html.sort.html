This is a method that sorts the child elements in ascending or descending order. It is called as follows:

wd(input).sort(order, col);

The order argument (optional) is the number that indicates the order. If less than zero, decreasing, otherwise increasing. The default value is 1.

The col argument (optional) is the number that indicates the column to use as a reference to define the order. If the child elements, which are to be sorted, also have children (columns), it is possible to define these elements as a parameter for sorting (eg td elements of a table). The default value is null.

//Example
function f() {
	var order = document.getElementById(js_html_sort_example1).value;
	wd$(#js_html_sort_example2).sort(order);
};
