dataWdPage

The purpose of the attribute is splits the children of the element into groups and display only a certain group.

See the following chapters:

  • WDdom: view (WDdom.page)

    Attribute

    data-wd-page=size{s}page{p}

    The subactions, page and size, have the same purpose as the same-named attribute of the HTML.page object.

    Target

    • Child elements.

    Sensitivity

    • After loading the document;
    • After defining the attribute on the element with the method WDdom.data;
    • After external data loading procedures using the WD Attributes Tools library;
