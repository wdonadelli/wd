The purpose of the attribute is displays only child elements, objects defined in the CSS selector, whose textual content (value/content) is contained in the element:

data-wd-filter=min:CSS

The min attribute is optional and has the same purpose as the same-named attribute of the HTML.filter object.

The CSS attribute is required and its value is the CSS selector that indicates the parent element to be filtered.

See the method HTML.filter to understand the source of its operation!

Target:

  • Element children defined by the CSS selector.

Sensitivity:

  • When loading the document.
  • When defining the attribute on the element (See the HTML.data method).
  • After external data loading procedures (repeat/load).
  • When you change the value of the form field from the keyboard
