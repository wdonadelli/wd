The purpose of the attribute is to load the contents of an accessible path or file into the element that contains it in HTML format:

data-wd-load=method:path

The method attribute is optional and the possible values ​​are get and post. The default value is get.

The path attribute is required and its value is the name of the accessible path or file to load.

See the methods HTML.load, Path.get and Path.post to understand the source of its operation!

Target:

  • Element content.

Sensitivity:

  • When loading the document.
  • When defining the attribute on the element (See the HTML.data method).
  • After external data loading procedures (repeat/load).

Example:

Sets Attribute:

data-wd-load= example.html?id=3 | post:example.txt |
