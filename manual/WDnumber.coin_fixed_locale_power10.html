WDnumber: text/html

WDnumber.coin

It is a method that returns a string with currency formatting if the browser contemplates the corresponding method. Otherwise, returns the number in a generic currency format, returning a String.

wd(number).coin(currency, locale)

The currency argument consists of a string representing the currency (e.g. USD to US Dollar). If not defined or non-existent, the entered text will be added in front of the return string that will be formatted according to the locale method.

The locale argument (optional) consists of a string representing the location (e.g. pt-BR). If not set, the location will be the one entered in the lang attribute of the html tag, the value provided by the browser or en-US.

Example

wd().coin(, );

wd().coin();

wd().coin();

WDnumber.fixed

It is a method that sets the minimum number of integers and the number of characters without numerical approximation, returning a String. The minimum number of integer characters and the number of decimal characters are respectively defined by the optional int and frac arguments.

wd(number).fixed(int, frac)

Example

wd().fixed();

wd().fixed(, );

wd().fixed(, );

WDnumber.locale

It is a method that returns a string with local numeric formatting, if the browser contemplates the corresponding method. Otherwise, it returns the number entered in string format.

wd(number).locale(locale)

The locale argument has the same purpose as the argument of the same name as the coin method.

Example

wd().locale();

wd().locale();

WDnumber.power10

It is a method that returns a string in scientific notation.

wd(number).power10(n, html)

The argument n (optional) indicates the number of decimal places to be displayed in the notation. The html argument (optional), if true, returns the value in html format, otherwise the return will be in text format.

Example

wd().power10();

wd().power10();

wd().power10(, true);
