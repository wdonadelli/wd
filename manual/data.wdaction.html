The purpose of the attribute is performs the action defined on elements identified by the CSS selector:

data-wd-action=action:CSS

The CSS attribute is required and its value is the CSS selector that identifies the elements where the action will be executed.

The action attribute is required and has the same purpose as the same-named attribute of the HTML.action object.

To enter more than one action, separate them with the & sign.

See the method HTML.action to understand the source of its operation!

Target:

  • Elements defined by the CSS selector.

Sensitivity:

  • When the element receives a click.
