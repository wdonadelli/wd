This is a method that returns the array with the item replaced by the new value. It is called as follows:

wd(input).replace(item, value);

The item argument is the value to be replaced and the value argument is the substitute value.

//Example
var x = [wd, 100, 100, wd];
wd(x).replace(100, 2);
[wd, 2, 2, wd]
//Variable "x" is not changed
