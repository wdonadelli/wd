    valueOf and toString methods

    The toString method returns the value of the array in JSON format. The valueOf method returns the browser default value.

    This is an object for number manipulation. The supplied argument (primitive value) is not changed by methods or attributes. The following primitive values ​​define the object:

    • Numbers
    • Numbers in string format.

    Numbers in text format will also be treated as numbers, so 5 and +5 are treated as 5, 1e2 and +1e+2 are treated as 100, +Infinity will be treated as Infinity, -Infinity as -Infinity and Infinity will not be treated as a number.

    When identifying the type of the primitive value, the object returns the string number.

    //Example
var input = 5.45;
var object = wd(input);
object.type;
number
