WDnumber

This is an object for manipulating numbers. The type attribute returns the value number. The following primitive values ​​define the object:

  • JavaScript Array Object
  • JavaScript String Object

In case of String, this must be in numeric format. To be considered a number, the numbers Infinity and -Infinity must be accompanied by the respective signs (+/-) so that the Strings can be considered a number.

    valueOf and toString methods

    The valueOf method returns the browser default value. The toString method returns the number in the form integer.decimal and the infinity symbol when entered.
