This is a method that executes the function entered in the argument in the same amount of existing elements, that is, if it is a list of nodes with n elements, the function will be performed n times. It is called as follows:

wd(input).execute(method);

The method argument is the function to be executed. The reported method accepts an argument representing the nth element.

//Example
function m(x) {
	x.style.color = #FF0000;
	x.textContent = Red Color;
};
wd$(#js_html_execute_example1, #js_html_execute_example2).execute(m);
