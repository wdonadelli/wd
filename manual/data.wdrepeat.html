The purpose of the attribute is to clone the child elements from a JSON format file to the parent element that contains the attribute:

data-wd-repeat=method:path

The method attribute is optional and the possible values ​​are get and post. The default value is get.

The path attribute is required and its value is the name of the JSON file/path to be loaded as a parameter for cloning child elements.

See the methods HTML.repeat, Path.get and Path.post to understand the source of its operation!

Target:

  • Child elements.

Sensitivity:

  • When loading the document.
  • When defining the attribute on the element (See the HTML.data method).
  • After external data loading procedures (repeat/load).
