This project consists of the distribution of two libraries for web application development.

The Web Development tool consists of a library written in JavaScript whose purpose is to provide various tools to give dynamism and agility to the application in development. This tool is divided into two segments, one based on the JavaScript code and another based on the attributes of the HTML elements.

The Web Design tool consists of a library written in CSS whose purpose is to provide the styling of the developing application by means of basic class attributes.

In order to demonstrate some of the functionalities of the libraries, this manual was constructed using the project tools.

The links to the libraries and the project are:

  • https://wdonadelli.github.io/wd/wd.css
  • https://github.com/wdonadelli/wd
