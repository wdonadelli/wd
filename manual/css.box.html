Row/Col



The wd-box-row class defines a container to group other containers formatted as columns of predefined sizes. The wd-box-col-XX sets the size of the columns of the wd-box-row child containers. The row supports up to 12 columns and each suffix represents how many cells will be occupied by the child container. For example, the -02 suffix will be two-cell size out of a total of 12. There are fewer cells than the total, so the columns will fit to fill the entire row. It does not have the suffix -child.
