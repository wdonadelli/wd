This is a method that defines actions for the element. It is called as follows:

wd(input).action(act);

The act argument is a string that represents the behavior to be applied to the element, as shown below:

open:
For the dialog element, executes the showModal() method, if defined; for the details element, sets the open attribute to true, if set; otherwise, it sets the data-wd-open attribute to the element.
close:
For the dialog element, executes the close() method, if defined; for the details element, sets the open attribute to false, if set; otherwise, it removes the data-wd-open attribute from element.
toggle:
Toggles between open and close.
tab:
Displays only the element informed, your brothers will be hidden.
del:
Removes the element from DOM.
show:
Shows the element.
hide:
Hides the element.
check:
Check input elements of type radio or checkbox.
uncheck:
Uncheck input elements of type radio or checkbox.
togglecheck:
Toggles the check of the input elements of the type radio or checkbox.
clean:
Clears the content of the element.
//Example
var x = document.getElementById(js_html_action_example1 > input);

function show() {
	wd$(x).action(show);
};

function hide() {
	wd$(x).action(hide);
};

function check() {
	wd$(x).action(check);
};
function uncheck() {
	wd$(x).action(uncheck);
};

function togglecheck() {
	wd$(x).action(togglecheck);
};
