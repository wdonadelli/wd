WD Style Sheet

The WD Style Sheet consists of a library written in CSS whose purpose is to provide the styling of the developing application by means of basic class attributes.

The tool relates slightly to the javascript tools resulting in integration between the two libraries.

Attribute

The class attribute starts with the prefix wd- followed by the style name:

wd-styleName

Depending on the style, there may be the suffix -child that transfers the attribute's style to the element's children:

wd-styleName-child
