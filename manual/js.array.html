This is an object for manipulating arrays. The supplied argument (primitive value) is not changed by methods or attributes. The following primitive values ​​define the object:

  • JavaScript Array Object

When identifying the type of the primitive value, the object returns the string array.

//Example
var input = [1, wd, true];
var object = wd(input);
object.type;

The object has the following attributes:

  • organized
  • sort
  • unique

The object has the following methods:

  • add
  • amount
  • del
  • replace
  • toggle
