This is a method that returns the primitive value plus the item entered in the argument, if nonexistent. It is called as follows:

wd(input).add(item);

The item argument is the value to be added to the array.

//Example
var x = [wd, 100, 100, wd];
wd(x).add(100);
[wd, 100, 100, wd]
wd(x).add(1);
[wd, 100, 100, wd, 1]
