This is a method that performs requests using the GET method and application/x-www-form-urlencoded enctype. It is called as follows:

wd(input).get(execute, time);

The execute argument is a function to be executed after request process. The function accepts an argument that returns an object containing four attributes:

error
Informs if an error occurred in the request (true/false).
request
Returns the request object.
text
Gets the result of the request in text format. Returns null in case of error.
xml
Gets the result of the request in xml format. Returns null in case of error.
json
Gets the result of the request in json format. Returns null in case of error.

The time argument (optional) is the value waiting time for the request, in seconds.

Example

//"file.php" is an accessible file on the server:
var object = wd(file.php?text=lux&number=12);
function y(x) {
	console.log(x.error);
	console.log(x.request);
	console.log(x.text);
	console.log(x.xml);
	console.log(x.json);
	return;
};
object.get(y, 10);

Use the expression @formName after the ? of the path so that the data of the elements of the specified form, by name attribute, are captured automatically, as if the form had been submitted. Only the data from the first form with the specified name will be obtained, do not use repeated names:

//Example
//"file.php" is an accessible file on the server and
//"form1" is the value of the "name" attribute of a "form" element:
var object = wd(file.php?@form1);

This is an object that executes and works with asynchronous requests. The supplied argument (primitive value) is not changed by methods or attributes. The following primitive values ​​define the object:

  • A file or path accessible on the server.

When identifying the type of the primitive value, the object returns the string path.

//Example
//"file.json" is an accessible file on the server:
var object = wd(file.json);
object.type;
path

The object has the following methods:

  • get
  • post

This is a method that performs requests using the POST method and application/x-www-form-urlencoded enctype. It is called as follows:

wd(input).post(execute, time);

This method works in the same way as the Path.get method. See the respective section for more details.

Example

function load(file) {
	wd(file).post(function(x) {
		wd$(#js_path_post_example2).get[0].textContent = x.error;
		wd$(#js_path_post_example3).get[0].textContent = x.request;
		wd$(#js_path_post_example4).get[0].textContent = x.text;
		wd$(#js_path_post_example5).get[0].textContent = x.xml;
		wd$(#js_path_post_example6).get[0].textContent = x.json;
		return;
	});
}
