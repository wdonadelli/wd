WDnumber: methods

WDnumber.coin

It is a method that returns the numeric value rounded.

wd(number).round(n)

The n argument (optional) indicates the number of decimal places to be considered in the rounding. If the argument is omitted, the number will be rounded to the integer above.

Example

wd().round();
