WD JavaScript Tools consists of a JavaScript object whose attributes and methods depend on the type and structure of the informed argument. As a rule, the methods do not change the input value.

Its syntax is as follows:

wd(input);

The input value defines the attributes and methods of the object. The value of input is organized into ten categories:

  • Array
  • Boolean
  • Date
  • HTML
  • Number
  • Path
  • RegExp
  • Text

See the menu for more details.

All methods and attributes returned by the wd object come from internal objects. With the exception of the object assigned to the "input" type path, whose inheritance comes from the object assigned to the input type text, other objects inherit their attributes and methods from a primitive object.

The primitive object has the following attributes:

  • type

The primitive object has the following methods:

  • valueOf
  • toString
  • toSource

Depending on the returned object, the attribute and methods listed above may have different behavior, whose characteristics will be presented individually.

type

Returns the type of the value entered in the input argument.It is called as follows:

wd(input).type;

The attribute may return the following values:

?
undefined values
0
null values and empty strings
bool
Boolean values
number
Numeric values
time
Time values
date
Date values
path
Accessible file path
text
Strings not specified
array
List values
regex
Regular expression
f()
Function object
object
Object values ({})
html
HTML element
[html]
List of HTML elements
doc
HTML Document
win
window object
unknown
Value returned when it is not possible to set the argument type, as above, or its constructor.

var x = document.querySelectorAll(li);
wd(x).type;
