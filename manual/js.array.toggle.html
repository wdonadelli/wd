This is a method that returns the array with the item entered, if it does not exist, or without the item entered, if it already exists. It is called as follows:

wd(input).toggle(item);

The item argument is the value to add or remove from array.

//Example
var x = [wd, 100, 100, wd];
wd(x).toggle(100);
[wd, wd]
wd(x).toggle(1);
[wd, 100, 100, wd, 1]
//Variable "x" is not changed
