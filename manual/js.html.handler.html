This is a method that sets event triggers to elements defined in input. It is a standalone method, it uses the corresponding attribute of the element to assign the methods to be executed when the event is triggered. It is called as follows:

wd(input).handler(events, remove);

The item argument must be an object whose keys match the events and the values ​​correspond to the methods. If there is more than one method to be defined in the event, these must be listed in an array. The methods will be executed in the sequence that they are informed in the list or as they are added. The null value deletes all methods of the event. Keys can be uppercase or lowercase and may or may not contain the prefix on (e.g onclick, click and cLicK work). You can not report repeated methods. If the event does not exist in the element, the method assignment is ignored.

The remove argument (optional) is a boolean value e his default value is false. If true, instead of adding methods to events, they will be removed.

Example

var x = document.querySelectorAll(a);
//adding events
wd(x).handler({
	onClick: method1,
	mouseenter: [method2, method3],
	ONMOUSELEAVE: method4
});
//removing methods from the event
wd(x).handler({
	onmouseenter: method3
}, true);
//removing all methods from the event
wd(x).handler({
	mouseenter: null,
	MOUSELEAVE: null
});
//removing all methods from the event and adding new
wd(x).handler({
	onclick: [null, method4, method1]
});
