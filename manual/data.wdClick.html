dataWdClick

The purpose of the attribute is to click on the element that contains it. This attribute is indicated to define an automatic click to the end of the external loads. For example, if you want to open a dialog box at the end of loading the document.

Attribute

data-wd-click=

Target

  • Element.

Sensitivity

    • After loading the document;
    • After defining the attribute on the element with the method WDdom.data;
    • After external data loading procedures using the WD Attributes Tools library.
