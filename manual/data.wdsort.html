The purpose of the attribute is to sorts the child elements from parent element that contains the attribute:

data-wd-sort=order

The order attribute has the same purpose as the same-named attribute of the HTML.sort object.

See the method HTML.sort to understand the source of its operation!

Target:

  • Child elements.

Sensitivity:

  • When loading the document.
  • When defining the attribute on the element (See the HTML.data method).
  • After external data loading procedures (repeat/load).

Example:

Sets Attribute:

data-wd-sort= 1 | -1

Result:

  • {{value}}
