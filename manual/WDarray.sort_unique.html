WDarray: organizing items

WDarray.sort

It is a method that returns the ordered array in an increasing manner obeying the following sequence of data types: null values, numbers, time, date, text and other information. There is an argument that, if true, excludes repeated information by changing those specific ones back to their default format.

wd(array).sort(unique)

Example

var x = ;
var y = wd(x);
y.sort();

y.sort(true);

WDarray.unique

It is a method that returns the array without repeating items. There is an argument that, if true, sorts the array items in the same order as the sort method (there is no change to the default format).
