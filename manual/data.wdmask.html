The purpose of the attribute is applies a mask to the content of the element if the value matches the regular expression, alerting the value of the content:

data-wd-mask=regex

The regex attribute is required and has the same purpose as the same-named attribute of the Regex.mask object.

See the method Regex.mask to understand the source of its operation!

Target:

  • Element content.

Sensitivity:

  • When loading the document.
  • When defining the attribute on the element (See the HTML.data method).
  • After external data loading procedures (repeat/load).
  • When you change the value of the form field from the keyboard
