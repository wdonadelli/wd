WDtime: setting time

WDtime.hour

It is a attribute that that gets/sets the value of the hours (0-24). If you set a value beyond the limits, the time will be adapted.

wd(time).hour

Example

var x = ;
var y = wd(x);
y.hour;

y.hour = -1;
y.hour;

y.hour = 25;
y.hour;
