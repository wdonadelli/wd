dataWdAction

The purpose of the attribute is performs the action defined on elements identified by the CSS selector.

See the following chapters:

  • WDdom (querySelectorAll)
  • WDdom: view (WDdom.action)
  • dataWdClick

Attribute

data-wd-action=act{target}

For various targets and subactions:

data-wd-action=act1{target1}act2{target2}

Target

  • Element defined by the CSS selector.

Sensitivity

  • When the element receives a click.
