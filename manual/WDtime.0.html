WDtime

This is an object for time manipulation, disregarding the effects inherent to the time zone. The type attribute returns the value time. The time should be a string in the following formats:

  • H:MM
  • H:MM:SS
  • HhMM *
  • H12:MM AM **
  • H12:MM PM **
  • * case insensitive
  • ** case insensitive and with optional white space
H
Hour with one or two digits (integer from 0 to 24).
MM
Minutes with two digits (integer from 00 to 59).
SS
Seconds with two digits (integer from 00 to 59).
H12
12h clock (integer from 1 to 12).

The special string %now specifies the current time. The textual representation of the time must be a valid value (00:00:00 to 24:59:59), otherwise it will not be treated as a time and a different object will be returned.

Obs.: 00:59:59 is equal to 24:59:59

    valueOf and toString methods

    The toString method returns the time in HH:MM:SS format. The valueOf method returns the number of seconds elapsed from 0 (00:00:00) until 86399 (23:59:59). If the time limit has been exceeded, plus or minus, the count will restart by observing the day cycle.
