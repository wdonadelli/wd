This is an object for boolean manipulation. The supplied argument (primitive value) is not changed by methods. The following primitive values ​​define the object:

  • false
  • true

When identifying the type of the primitive value, the object returns the string bool.

//Example
var input = true;
var object = wd(input);
object.type;
bool

The object has the following attributes:

  • value (Deprecated - js.v.1.2.0 - 2019-03-20)

The object has the following methods:

  • valueOf
