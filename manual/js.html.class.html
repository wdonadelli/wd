This is a method that manipulates the class attribute. It is called as follows:

wd(input).class(list);

The list argument is an object that accepts three keys: add, del and toggle, executed in this order. The key values ​​are strings that represent the value to be added, removed, or alternate, respectively. If there are more than one attribute to set, the values ​​must be contained in an array. If the value is null, the class attribute is empty..

//Example
var x = document.querySelectorAll(a);
//Clearing the attribute
wd(x).class(null);
//Adding item to attribute
wd(x).class({
	add: class1
});
//Adding items to attribute
wd(x).class({
	add: [class2, class3, class4]
});
//Removing items from the attribute
wd(x).class({
	del: [class3, class4]
});
//Toggling attribute items
wd(x).class({
	toggle: [class1, class5]
});
//class1 has been removed and class5 added
