This is a method that manipulates the style attribute. It is called as follows:

wd(input).style(list);

The list argument is an object whose keys and values ​​represent, respectively, CSS attributes and values. Keys can be in CSS or camelCase style. If the attribute does not exist, it will be ignored.

//Example
var x = document.querySelectorAll(a);
wd(x).style({
	backgroundColor: red, //camelCase
	z-index: 2,           //CSS
	color: null             //removing attribute
});
