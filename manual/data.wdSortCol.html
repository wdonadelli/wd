dataWdSortCol

The purpose of the attribute is to sort table columns (table tag). The attribute must be contained in the elements td or th, children of tr, child of thead. When the element with the attribute is clicked, its column will be sorted in ascending order. If clicked again, the sort order will be reversed.

See the following chapters:

  • WDdom: view (WDdom.sort)
  • dataWdClick

Attribute

data-wd-sort-col=

Target

  • tbody children.

Sensitivity

  • When the element receives a click.
