This is a attribute that returns the value of the string with the first letter of each word in uppercase. It is called as follows:

wd(input).title;

Example

var x = DEUS lux mundi est.;
wd(x).title;
DEUS Lux Mundi Est.
wd(x.toLowerCase()).title;
Deus Lux Mundi Est.
//Variable "x" is not changed

This is a attribute that returns the value of the string using the title and trim attributes. It is called as follows:

wd(input).tt;

Example

var x =     Deus        lux     mundi est.      ;
wd(x).tt;
Deus Lux Mundi Est.
//Variable "x" is not changed

This is a attribute that returns the value of the string without unnecessary spaces. It is called as follows:

wd(input).trim;

Example

var x =     Deus        lux     mundi est.      ;
wd(x).trim;
Deus lux mundi est.
//Variable "x" is not changed
