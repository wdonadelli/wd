WDtext: formatting

WDtext.clear

It is a method that removes the accents from the alphabet. If the change argument is true, the informed text will change and the method will return itself, allowing chain changes. Otherwise, the method returns the changed value only, without changing the internal value.

wd(text).clear(change)

Example

var x = ;
var y = wd(x);
y.clear();

y.toString(); //did not change the original text

y.clear(true).toString();//changed the original text (application of the chain method)

WDtext.replace

It is a method that replaces the information contained in the first argument (old) with the information contained in the second argument (new). All identical elements located in the text will be replaced. The change argument has the same purpose as the argument of the same name as the clear method.

wd(text).replace(old, new, change)

The new argument accepts regular expressions but, if null or undefined, nothing will be substituted. The argument old, if null or undetermined, will have an empty string defined as value. If the override changes the original value and results in a different text value, an apostrophe is added to the beginning of the value.

Example

var x = ;
var y = wd(x);
y.replace(deus, Deus);

y.replace(mundi);

y.replace(lux, LUX, true).replace(Mundi, MUNDI, true).replace( , _);

wd(x).replace(/./, 8, true).toString();

WDtext.title

It is a method that changes the value of String by leaving only the first letter of each word in uppercase. The change argument has the same purpose as the argument of the same name as the clear method.

wd(text).title(change)

Example

wd().title();

WDtext.trim

It is a method that changes the value of String without unnecessary spaces. The change argument has the same purpose as the argument of the same name as the clear method.

wd(text).trim(change)

Example

wd().trim(true).replace( , -);

wd().title(true).trim();
