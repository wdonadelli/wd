dataWdData

The purpose of the attribute is to define the dataset method of the elements indicated by the CSS selector.

See the following chapters:

  • WDdom (querySelectorAll)
  • WDdom: setting (WDdom.data)
  • dataWdClick

Attribute

data-wd-data=key{value}${target}

For various targets and subactions:

data-wd-data=key1{value1}key2{value2}${target}&key3{value3}${target3}

Target

  • Element defined by the CSS selector.

Sensitivity

  • When the element receives a click.
