This is a method that displays only child elements that display the textual content entered in the argument. It is called as follows:

wd(input).filter(text, min);

The text argument is the text to be located.

The min argument (optional) is the minimum number of characters for the method to be executed. The default value is zero.

//Example
function f() {
	var text = document.getElementById(js_html_filter_example1).value;
	var min  = document.getElementById(js_html_filter_example2).value;
	wd$(#js_html_filter_example3).filter(text, min);
};
