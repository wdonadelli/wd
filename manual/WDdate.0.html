WDdate

This is an object for date manipulation, disregarding the effects inherent to the time zone. The type attribute returns the value date. The date should be a string in the following formats:

  • YYYY-MM-DD
  • MM/DD/YYYY
  • DD.MM.YYYY
YYYY
Four-digit year (integer from 0001 to 9999).
MM
Month with two digits (integer from 01 to 12).
DD
Day with two digits (integer from 01 to 31).

The Date object will also be considered as the date. The special string %today specifies the current date. The textual representation of the date must be a valid date, otherwise it will not be treated as a date and a different object will be returned.

    valueOf and toString methods

    The toString method returns the date in YYYY-MM-DD format. The valueOf method returns the number of days elapsed from 0001-01-01 (day 1) until day 9999-12-31 (last day), based on the projection of the Gregorian calendar. If the date limit is exceeded, the date will be set at the respective limit.
