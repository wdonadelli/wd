dataWdMask

The purpose of the attribute is to mask the content of the element if its value matches the regular expression defined.

See the following chapters:

  • WDregexp: mask

Attribute

data-wd-mask=RegExp

Target

  • Element content.

Sensitivity

  • After loading the document;
  • After defining the attribute on the element with the method WDdom.data*;
  • After external data loading procedures using the WD Attributes Tools library;
  • When you change the value of the form field from the keyboard.

* The textual content of the element will not change when another mask is already applied.
