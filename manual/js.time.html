This is an object for time manipulation, disregarding the effects inherent to the time zone. The time record can be changed by methods and attributes, the original value is not changed. The following primitive values ​​define the object:

  • H:MM
  • H:MM:SS
  • H12:MMam (case insensitive)
  • H12:MMpm (case insensitive)
  • H24hMM (case insensitive)
  • %now

Where:

H:
Hour (integer from 0).
MM:
Minutes with two digits (integer from 00 to 59).
SS:
Seconds with two digits (integer from 00 to 59).
H12:
12h clock (integer from 1 to 12).
H24:
24h clock with two digits (integer from 00 to 23).
%now:
Current time.

When identifying the type of the primitive value, the object returns the string time.

//Example
var x = wd(1:27);
x.hour;
1
x.hour = 2;
x.hour;
2
x.toString();
02:27:00
x.toSource();
1:27
x.type;
time

The object has the following attributes:

  • hour
  • minute
  • second
  • h24
  • ampm

The object has the following methods:

  • toString
  • valueOf
  • format
