WDdom.getStyle


		
It is a method that returns the computed style of the element. The css argument is the name of the style attribute to search. The function return is an array whose items correspond to the values ​​of the attribute indicated in the argument for each element. It does not have self-return behavior.

WDdom.item


		
It is a method that returns the element contained in the element set from its numeric reference information.


		
wd(dom).item(index)


		
The index argument indicates the element to return. If the number is less than zero, returns the element from the last element (-1 returns the last, -2 the second to last, and so on). If the value is not numeric, returns an array with all elements.


		
Example


		
HTML code


		


		
JavaScript

	
		
wd$(#eg11 > *).item().textContent;



		
Result


		

			
Text 1

			
Text 2

			
Text 3
