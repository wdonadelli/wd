dataWdLoad

The purpose of the attribute is to load the contents of an accessible path or file into the element that contains it in HTML format.

See the following chapters:

  • WDdom: content (WDdom.load)
  • WDtext: request

Attribute

For the POST method (default):

data-wd-load=post{HTML file path}

For the GET method:

data-wd-load=get{HTML file path}

Target

  • Element content.

Sensitivity

  • After loading the document;
  • After defining the attribute on the element with the method WDdom.data;
  • After external data loading procedures using the WD Attributes Tools library.
